# Landing Page Project

This project is part of [The Odin Project](https://www.theodinproject.com) curriculum. The goal is to create a landing page using HTML and CSS.

## Project Description

The landing page will include the following sections:
- Header with navigation links
- Hero section with a prominent call-to-action
- Features section highlighting key aspects
- Footer with contact information and social media links

## Technologies Used

- HTML
- CSS
